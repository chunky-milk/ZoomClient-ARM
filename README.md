# Zoom Client for Arm
using [box86](https://github.com/ptitSeb/box86/) or [box64](https://github.com/ptitSeb/box64/), depending on your system architecure.

### Installation
Open a terminal, execute this one command:
```
wget -qO- https://git.io/J8asc | bash
```
